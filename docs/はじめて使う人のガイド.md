## 📌 まず大事なこと

このリポジトリを使うために、**Gitの知識は必要ありません**。

「Gitでダウンロード」という発想を捨ててください。
実際にやることは、**たった3行のコマンドをコピペするだけ**です。

「えっ、それでいいの?」と思った方、その通りです。

# 🚀 Step 1：Claude Code をインストール

## やること

[claude.com](https://claude.com) にアクセスして、Claude Code をインストール。

## 詳しい手順

1. ブラウザで https://claude.com を開く
2. 右上の「Sign up」または「Try Claude Code」をクリック
3. メールアドレスでアカウント作成
4. 「Download Claude Code」をクリック
5. お使いのOS（Mac / Windows）に合わせてダウンロード
6. ダウンロードしたファイルを開いてインストール

## 不安なポイント

**Q. お金かかる?**
無料体験あり。本格利用は月額20ドル〜。最初は無料で十分試せます。

**Q. インストール中にエラーが出た**
ブラウザを変える（Chrome推奨）、または再起動して再試行。

---

# 📂 Step 2：ターミナルを開く

## やること

**ターミナル**（黒い画面）を開きます。

## Macの場合

1. キーボードで `Command + Space` を押す
2. 検索欄に「ターミナル」と入力
3. 黒いアイコンの「ターミナル」をクリック

## Windowsの場合

1. スタートメニューを開く
2. 「PowerShell」または「コマンドプロンプト」と入力
3. クリックして開く

## 画面に出るもの

こんな感じの黒い（または白い）画面が出ます：

```
ryou@MacBook ~ %
```

最後の `%` または `$` が**「ここに打ち込んでね」のしるし**です。

## 不安なポイント

**Q. 黒い画面、怖いです**
プログラマーが普段使う画面です。**間違えても壊れません**。Enterを押さなければ何も起こらないので、ゆっくり進めてOK。

---

# 🎯 Step 3：ワンライナーを貼り付ける

## やること

下のコマンドを**1行コピペ**してEnter。

```bash
curl -fsSL https://raw.githubusercontent.com/Tagaru05/tagaru-study-hub/main/install.sh | bash
```

## 詳しい手順

1. 上のコードブロックの**右側にコピーボタン**があります（GitHubで見ている場合）
2. クリックしてコピー
3. ターミナルにペースト：
   - Mac: `Command + V`
   - Windows: 右クリック or `Ctrl + V`
4. Enterキーを押す

## 何が起きるか

ターミナルに、こんな表示が流れます：

```
🎓 タガル式・自分の教科書 セットアップ
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

▶ Step 1/5: Python の確認...
✅ Python OK: Python 3.x.x

▶ Step 2/5: pip の確認...
✅ pip OK: pip3

▶ Step 3/5: notebooklm-py をインストール中...
（インストールのログが流れる）
✅ notebooklm-py インストール完了

▶ Step 4/5: NotebookLM ログイン
   ブラウザが自動で開きます...
```

途中で **ブラウザが自動で開きます**。次のステップで対応します。

## 不安なポイント

**Q. 「Python が見つかりません」と出た**
Macなら最初から入っているはずですが、出たら：
- Mac: [python.org](https://python.org) から最新版をダウンロード
- Windows: 同上。インストール時に「Add to PATH」にチェック

**Q. 「permission denied」と出た**
コマンドの前に `sudo` を付けて再実行。パスワードを聞かれます。

# 🔑 Step 4：Googleログイン

## やること

自動で開いたブラウザで、Googleアカウントにログインします。

## 詳しい手順

1. ブラウザが自動で開く
2. 「Sign in with Google」をクリック
3. 普段使っているGoogleアカウントを選ぶ
4. 「許可（Allow）」をクリック
5. 「Authentication successful」と表示されたらブラウザを閉じてOK

## なぜGoogleログインが必要?

NotebookLMがGoogleのサービスだからです。
あなたのアカウントでNotebookLMを操作する権限を、Claude Codeに渡すための一回限りの手続きです。

## 不安なポイント

**Q. 個人情報、大丈夫?**
このログインは、あなたのパソコンとGoogleの間だけで完結します。
第三者にあなたのパスワードや情報が渡ることはありません。

**Q. 「アクセス権を要求しています」が怖い**
要求されるのは「NotebookLMへのアクセス」だけ。GmailやDriveには触れません。

# 🎮 Step 6：実際に使ってみる

## やること

Claude Code を起動して、テーマを伝えるだけ。

## 詳しい手順

1. Claude Code を起動
2. プロジェクトを開く（どのフォルダでもOK・最初は `Desktop` でいい）
3. メッセージ欄に以下を入力：

```
/study-hub 学びたいテーマ
```

例：
- `/study-hub 感情`
- `/study-hub マーケティング`
- `/study-hub 自己肯定感`
- `/study-hub 人類の歴史`
- `/study-hub 量子力学`

## 何が起きるか

Claude Code が、約1〜2時間かけて：

1. テーマを観点に分解する（あなたに確認します）
2. 4人のAIで並列リサーチ
3. 9章の教科書を組み立て
4. NotebookLMに自動アップロード
5. マインドマップ・音声・FAQを生成

完了したら、ブラウザでNotebookLMを開くと、**あなた専用の学習ハブ**が完成しています。

# 🚨 トラブルシューティング

## install.sh で止まった

### 「command not found: pip」
Pythonが入っていません。[python.org](https://python.org) から最新版をインストール → 再実行。

### 「Permission denied」
コマンドの前に `sudo` を付ける：
```bash
sudo curl -fsSL https://raw.githubusercontent.com/Tagaru05/tagaru-study-hub/main/install.sh | bash
```

### Googleログイン中にエラー
- ブラウザのキャッシュをクリア
- 別のブラウザ（Chrome推奨）で再試行
- それでもダメなら：`notebooklm auth check` で診断

## Claude Code側で動かない

### `/study-hub` が認識されない
- Claude Codeを一度再起動
- スキルがインストールされたか確認：
  ```bash
  ls ~/.claude/skills/study-hub/
  ```
  ファイルがあればOK

### リサーチが途中で止まる
- Claude Codeを再起動して再試行
- ネット接続を確認

### NotebookLM音声生成が失敗
- Googleのレート制限の可能性
- 5〜10分待って再試行

# 💬 よくある不安への回答

## 「私はプログラミングできないんですが…」
このガイドの通りに、コピペだけで進められます。
中で起きていることを理解する必要はありません。
**料理のレシピを「分子レベルで理解する必要がない」のと同じ**です。

## 「途中で失敗したら全部壊れません?」
**壊れません**。失敗してもパソコンには何の影響もありません。
最悪、何度でもやり直せます。

## 「変な英語が画面に流れて怖い」
英語のメッセージは、**「いま何をしているか」の実況中継**です。
読まなくて大丈夫。`✅` マークと `🎉` マークだけ見ていればOK。

## 「インストールしたものを後で消したい」
以下のコマンドで全部消せます：
```bash
pip uninstall notebooklm-py
rm -rf ~/.claude/skills/study-hub/
rm -rf ~/.notebooklm/
```

## 「教科書を作ったら、それは公開されちゃう?」
**公開されません**。
- Markdownファイル：あなたのパソコンの中だけ
- NotebookLM上：あなたのGoogleアカウント内だけ
- 共有したい時だけ、自分で共有設定をすればOK

## 「他のテーマでも何度でも使えますか?」
**使えます**。テーマを変えて `/study-hub` を実行すれば、毎回新しい教科書が作られます。
