## 💰 料金について

### Q. 全部でいくらかかりますか？

| サービス | 料金 |
|---|---|
| Claude Code | 月額20ドル〜（無料体験あり） |
| NotebookLM | 無料プランあり（Pro月額2,900円〜） |
| notebooklm-py | 完全無料（GitHub公開） |
| このスキル | 完全無料（MITライセンス） |

最小構成なら、**Claude Code の無料体験のみで試せます**。

### Q. NotebookLMの無料プランで足りますか？

最初は十分です。1ノートブックあたり50ソースまで投入可能。
頻繁に使うなら Plus（100ソース）以上を検討してください。

## 🔧 セットアップについて

### Q. プログラミング知識は必要？

**ほぼ不要**です。`install.sh` のワンライナーをコピペできれば動きます。
あとは日本語でAIに話しかけるだけ。

### Q. Macじゃないとダメ？

Windows・Linuxでも動きます。Claude Code が対応しているOSなら使えます。

### Q. ターミナルが怖いです

最初の一回だけです。`install.sh` を貼って Enter を押すだけ。
普通のアプリのインストールと変わりません。

### Q. install.sh の中身が分かりません

中身は[こちら](../install.sh)で全部見られます。
やっていることは「NotebookLMのリモコンをインストール → ログイン → スキルを配置」だけです。

## 🎓 使い方について

### Q. どんなテーマで使える？

何でも。ただし「自分が本気で知りたいこと」が向いています。
例：
- 学問（哲学・心理学・歴史など）
- 仕事のテーマ（マーケティング・営業・リーダーシップ）
- 個人のテーマ（自己肯定感・人間関係・お金）

### Q. 教科書ができるまでどれくらいかかる？

| 段階 | 時間 |
|---|---|
| 観点分解 | 30秒 |
| 並列リサーチ | 5〜10分 |
| 教科書執筆 | 30〜90分 |
| NotebookLM投入 | 2〜5分 |
| 成果物生成（マインドマップ・音声等） | 10〜20分 |

**実時間：合計1〜2時間。** ただし、ほとんどはAIが裏で動いているので、あなたは別のことをしていてOK。

### Q. 完成した教科書はどこに残る？

- **Markdownファイル**：あなたのMacの中（永久保存）
- **NotebookLM**：Google上のあなたのアカウント
- どちらも消えません

### Q. 自分の作った教科書を売っていい？

もちろんOK。教科書はあなたが作ったものなので、あなたのもの。
ただし、このスキル本体（SKILL.md）を再配布する場合はMITライセンスに従ってください。

## 🤔 哲学について

### Q. ChatGPTで質問するのと何が違う？

ChatGPTで質問する → **ChatGPTの言葉**が返ってくる。
このメソッド → **自分の言葉**が育つ。

教科書を作る過程で、自分の中で言語化のプロセスを通るからです。

### Q. 「目的は教科書を作ることじゃない」ってどういう意味？

教科書は**副産物**です。
本当の目的は、**学ぶ過程で自分が変わること**。
だから、AIに「全部やらせて完成品だけ受け取る」のは推奨しません。
必ず章末問題を自分で書き出してください。

### Q. AIに「使われる」と「使う」の違いは？

- 使われる：AIに質問して、AIの答えを読む
- 使う：AIに自分の代わりに研究させ、組み立てさせ、自分が学べる場所まで作らせる

このメソッドは後者です。

## 🐛 トラブル

### Q. `notebooklm login` でエラーが出る

- ブラウザを一度閉じてから再試行
- `notebooklm auth check` で診断

### Q. 並列リサーチが途中で止まる

サブエージェントの権限拒否の可能性。Claude Code を再起動して再試行。

### Q. NotebookLMの音声生成が失敗する

Googleのレート制限の可能性。5〜10分後に再試行してください。
