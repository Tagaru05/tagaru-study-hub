## 🛡️ 相手にトラブルが起きたとき

Claude Code Desktop の最大の強みは、**エラーが出たらそのまま Claude に直してと頼める**こと。

相手から「○○できない」と言われたら、まずこの呪文を案内：

```
上のインストールがうまくいきませんでした。
エラーを見て、原因を直してから、もう一度インストールを最後まで進めてください。
途中で必要なソフト（PythonやHomebrew等）が足りなければ、それも入れてください。
```

これだけで、相手の Claude Code が：
1. エラーログを読む
2. 原因を特定する
3. 解決コマンドを提案して実行する
4. インストールを最後まで通す

それでも解決しない場合の案内：

| 相手の状態 | あなたが案内するもの |
|---|---|
| そもそも何をしていいか分からない | [はじめて使う人のガイド](はじめて使う人のガイド.md)、または[統合HTMLサイト](https://tagaru05.github.io/tagaru-study-hub/) |
| よくある質問・不安 | [FAQ](FAQ.md) |
| 仕組みを詳しく知りたい | [仕組み解説](仕組み解説.md) |
| 「Claude Code 自体がわからない」 | https://claude.com の Sign up からアカウント作成 → Download |

それでも解決しなければ、**スクリーンショットを送ってもらう**のが最速です。
Claude Code のチャット画面を画像で送ってもらえば、どこで止まっているか一発で分かります。

- 「ガイドの中の青いコピーボタンの文章を貼ってね」で十分

- LINEテンプレ（上記1️⃣）を送る

- 一言：**「コピペするだけだから大丈夫」**
- 「料理のレシピを分子レベルで理解する必要がないのと同じだよ」と伝える
- できれば**最初の1回は画面共有で一緒にやってあげる**のが一番確実
- セットアップさえ越えれば、あとは日本語で話しかけるだけ
