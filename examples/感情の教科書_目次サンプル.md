# 実例：感情の教科書（目次サンプル）

これは「タガル式・自分の教科書」で最初に作られた教科書の目次です。
2026年5月、テーマ「感情」で実行した結果、9章構成の教科書が完成しました。

| 項目 | 内容 |
|---|---|
| テーマ | 感情とは何か |
| 観点 | 科学・哲学・言語化・実践（4本） |

## 9章の章立て

```
序章｜なぜ感情を学ぶのか
  → 学ぶ理由・出発点記録ワーク

第1章｜感情とは何か
  → 定義の歴史と現代の到達点
  → 3つの立場（本能論／身体反応論／構成主義）

第2章｜感情の正体 — 身体・脳・予測
  → 3階建ての構造
  → 脳の主役3つ（扁桃体・前頭前野・島皮質）

第3章｜感情の地図 — 種類と分類
  → エクマン6基本感情〜Cowen & Keltner 27カテゴリ
  → 日本独自の感情語（甘え・もののあはれ）

第4章｜感情と言葉
  → アフェクトラベリングの脳科学
  → 感情粒度（emotional granularity）

第5章｜感情を読む技術 — 観察と命名
  → 3ステップ（気づく→観察する→名づける）
  → 自分専用の感情マップ作成

第6章｜感情と付き合う技術 — 7つのアプローチ
  → EQ／マインドフルネス／ACT／CBT／IFS／NVC／ジャーナリング

第7章｜感情とAI — 新しい時代の感情リテラシー
  → AI時代の新感情カテゴリ
  → AIを「感情の鏡」として使う

終章｜自分の言葉で語る
  → 統合と最終ワーク

付録｜索引と用語集
  → 五十音逆引き・人物年表
```

---

## 各章に入っている5パーツ

すべての章に、以下の5つが含まれます：

| パーツ | 内容 |
|---|---|
| 🎯 この章のゴール | 読み終わった時に何が言えるようになるか |
| 📖 本文 | 知識の解説・図解・たとえ込み |
| 🎬 日常のシーン | 「あの時のあれ」と接続するエピソード |
| ✍️ ワーク | 自分で書き出す問い |
| 📝 章末問題 | 自分の言葉で答えるチェック |
